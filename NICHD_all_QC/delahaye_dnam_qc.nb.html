1.2.1 Get meth/unmeth matrices

Often, raw DNAme data on GEO or from Genome Studio is stored in the form of a dataframe with columns containing the methylated and unmethylated (or A/B or Red/Green) signal intensities. This can be parsed into a methylset, which is a minfi object built to contain these intensity values.

According to Page 48 of the the Genome Studio Methylation Module v1.8 User Guide, Methylated == Signal B == Green

This data is often stored seperately as a supplementary file in GEO.

Non-specific probes

Based on the work of Magda (Price), we know that several probes on the 450K and EPIC arrays are non-specific to their target locations in the genome, that is, based on sequencing similarity they have been predicted to bind to multiple locations in the genome and are therefore cross-hybridizing and/or nonspecific. These probes are therefore not reliable, as we dont know which location in the genome they are actually reflecting the methylation patterns of, and we remove these probes prior to analysis.

Polymorphic

Probes with SNPs (>1% MAF) in the SBE site of the CpG of interest will be filtered using both Chen et al.2013s annotation for the array (from the Weksberg lab) and Kasper D. Hansens built in SNP data for the Illumina arrays. Information on Hansens data can be found in the package pdf for the Illumina manifest. These objects are named SNPs.###CommonSingle with ### corresponding to the version of dbSNP for which the data was constructed, higher numbers represent later versions of dbSNP.

Chen et al.2013: https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3592906/

Chen annotation can be downloaded from the Weksberg lab website: http://www.sickkids.ca/Research/Weksberg-Lab/Publications/index.html * (this URL annoyingly sometimes changes, google Weksberg lab if you cant locate it, usually under their Publications page)

Hansen: https://bioconductor.org/packages/release/data/annotation/manuals/IlluminaHumanMethylation450kanno.ilmn12.hg19/man/IlluminaHumanMethylation450kanno.ilmn12.hg19.pdf NOTE This will NOT be applied since we are interested in the association of SNP and DNA methylation.

4.2.1 minfi::Quantile

The minfi function for quantile normalization is applicable to both rgest and msets. For a description of this method, see Touleimat & Tost, 2012. This function returns a GenomicRatioSet. This method relies on a reference distribution of Type I probes that it normalizes the distribution of Type II probes against, so setting the seed is important to ensure the same random distribution of Type I reference probes is selected each time, for reproducible normalization.

https://www.rdocumentation.org/packages/minfi/versions/1.18.4/topics/preprocessQuantile
